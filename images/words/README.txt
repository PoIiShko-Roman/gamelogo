Помістіть сюди ілюстрації до слів для вагончиків.
Рекомендації:
- створіть підпапку з назвою літери (наприклад, "а", "б"), як у файлі js/words.js;
- формат SVG або PNG 512×512;
- назва файлу збігається з ключем слова у файлі js/words.js (наприклад, apple.png);
- для порожніх позицій можна тимчасово використовувати заглушку fallback.svg з папки images/game-cards.
